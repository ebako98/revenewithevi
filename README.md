# revenewithevi.com

Coming soon page for [revenewithevi.com](https://revenewithevi.com).

Built with plain HTML, CSS and JavaScript. Hosted on GitHub Pages.

## Setup

1. Replace `YOUR_FORM_ID` in `index.html` with your [Formspree](https://formspree.io) form ID to enable contact form submissions.
2. Push to GitHub and enable GitHub Pages in repository settings.
3. Add custom domain in GitHub Pages settings and update DNS records on GoDaddy.
